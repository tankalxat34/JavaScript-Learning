## Объекты

**Главная идея в JavaScript: все сущности являются объектами**.

**Объект** - набор свойств "имя: значение".

Пример объекта:

```js
{
    visible: true,
    colorDepth: 24,
    title: 'My Image',
    orient: {
        angle: 0,
        type: 'landscape'
    },
    pixelDepth: 24,
    width: 1440,
}
```

Массивы, функции, числа, строки - все эети сущности ведут себя как объекты.
